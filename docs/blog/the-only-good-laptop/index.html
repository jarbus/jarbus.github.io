THE ONLY COMPUTER I NEED

I just upgraded the processor in my Framework 13, and all things considered, it went well. Most importantly, it was easy. I also upgraded the webcam, which took about five minutes to swap.

The original machine with a Ryzen 7640U was $1k, and the HX 370 mainboard upgrade was $1k. You could probably get a “better” laptop for $2k, but it wouldn’t be nearly as repairable or upgradable. You’d also only have a single computer. With a framework upgrade, you get two machines.

There’s a lot of cool DIY projects people do with their old framework mainboards, but I didn’t expect to find a use for mine. We ended up turning it into a personal server using the $40 Coolermaster Case. Our previous server was a ten-year-old budget gaming machine that used way more power and had louder fans, so this was a welcome upgrade.

My “old” mainboard is a wonderful server. It’s tiny, quiet, and draws little power, as you might expect from a 13-inch laptop motherboard. We’re only running two docker services, which the mainboard is more than capable of handling.

Re-using old hardware also feels nice. I’m getting legitimate value out of repurposing something I already have. I didn’t buy a whole new laptop and I didn’t throw out an old one (excluding the original webcam).

Honestly, I can’t see myself buying a new laptop as long as Framework’s keeps making mainboards that are compatible with the current 13-inch chasis.
